Zadanie 5

Środowisko deweloperskie to środowisko, w którym programista tworzy, testuje i eksperymentuje z modelem uczenia maszynowego.  
Najczęściej działa ono lokalnie na komputerze programisty lub w środowisku testowym.
cechy środowiska deweloperskiego:
- praca na mniejszych zbiorach danych
- częste eksperymentowanie z różnymi algorytmami
- brak dużych wymagań dotyczących wydajności
- możliwość ręcznego uruchamiania kodu i testów
- brak konieczności ciągłej dostępności systemu

przykład:
Trenuje model lokalnie w Pythonie i sprawdzam jego accuracy na zbiorze testowym.


Środowisko produkcyjne to środowisko, w którym model jest wykorzystywany przez użytkowników lub inne systemy.  
Model musi działać stabilnie, szybko i być dostępny przez cały czas.
cechy środowiska produkcyjnego:
- obsługa dużej liczby zapytań
- wymagania dotyczące wydajności i stabilności
- konieczność monitorowania działania modelu
- automatyczne wdrażanie nowych wersji
- zarządzanie wersjami modeli i zależnościami

przykład:
Model działa jako API, które przewiduje wynik dla danych przesyłanych przez aplikację.
